# Keyboards

Reference for my two ergo keyboards (kept in different countries), so I can
cross-check mappings when I'm away from one of them. The goal is to keep their
layouts as close as possible.

## Kinesis Advantage360 (SmartSet)

- `kinesis-360/layout1.txt` - full remap, annotated with comments per block.
- `kinesis-360/led1.txt` - LED indicators (keypad = white, layer 1 = blue,
  layer 2 = green, layer 3 = red).

To apply: put the keyboard in v-drive mode (SmartSet), copy these files over
the ones in the `active` folder, then eject/reload.
